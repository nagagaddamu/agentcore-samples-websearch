# Set Up Inbound Authorization

Configure authorization for inbound requests to your Amazon Bedrock AgentCore gateway. Control which agents and users can access specific tools through HTTP and MCP protocols.

## Tutorials

| Section                 | Description                                              |
| :---------------------- | :------------------------------------------------------- |

| [mcp](mcp/)             | Set up inbound authorization for MCP tool targets        |
| [inference](inference/) | Set up inbound authorization for inference (LLM) targets |
